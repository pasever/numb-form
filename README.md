# numb-form

Number Formatter
=========

A small library that adds commas to numbers

## Installation

  `npm install numb-form`

## Usage

    const numFormatter = require('@pasever/numb-form');

    const formattedNum = numFormatter(67890);
  
  
  Output should be `67,890`


## Tests

  `npm test`
